# Lost and Found Portal

A web-based application that helps students report and find lost items within a campus environment.
Users can post details of lost or found items, making it easier for owners to recover their belongings.

## Features

* Post information about **lost items**
* Post information about **found items**
* View a list of reported items
* Simple and responsive user interface
* Page Not Found (404) handling

## Tech Stack

* **Backend:** Node.js, Express.js
* **Frontend:** HTML, CSS, EJS
* **Version Control:** Git

## Installation

1. Clone the repository

```
git clone https://github.com/Jaspreet1117/Lost-And-Found-Portal.git
```

2. Navigate to the project folder

```
cd Lost-And-Found-Portal
```

3. Install dependencies

```
npm install
```

4. Start the server

```
node index.js
node --watch index.js
node start
```

5. Open in browser

```
http://localhost:3000
```
